# idv-jumpdash
适用于第五人格2026年限时活动织梦兔的礼物中跳一跳小游戏的脚本

使用AutoHotKey v2，claude辅助编程
请使用管理员模式运行，否则无法在游戏中正常模拟鼠标按下

使用方法

1 【初次使用】按下f6来创建autojump.ini，这个自动检测不准确，建议手动编辑autojump.ini修改ms_per_px，对于游戏默认分辨率（1334*750），适用的ms_per_px值为2

2 进入跳一跳界面后，鼠标移动到奥菲所在方格的中心，按下f2，移动鼠标到目标方格的中心，松开f2，脚本开始模拟鼠标按下与松开

3 脚本模拟期间请不要移动鼠标&按下鼠标
